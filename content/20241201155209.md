# フルーツトマトの冷製パスタ

## 材料

- 元の動画だとバルサミコ酢とか入れてるけど、大体家にないので無視する
    - 多分入れたほうが上手い

|材料                      |量    |
|--------------------------|------|
|フルーツトマト            |2, 3個|
|ニンニク                  |極少量|
|バジル                    |      |
|オリーブオイル            |      |
|100%みかんジュース        |      |
|塩                        |      |

## 作り方

- フルーツトマト4等分, ニンニクちょびっと, バジル2,3枚細く切る
- オリーブオイル、みかんジュース入れて混ぜる
- 混ぜたら塩入れる
- 冷やした方が美味いので冷蔵庫入れておく
- パスタ茹でる
- 氷水に塩1.5%入れて、パスタ入れる
- パスタの水気を結構しっかり切る
- 皿にトマト先に盛り付け
- 残ってる汁に追いオリーブオイルかける
- 皿に上からジャーっとしたら完成
